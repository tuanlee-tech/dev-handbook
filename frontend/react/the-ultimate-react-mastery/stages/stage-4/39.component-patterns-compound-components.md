### 1.1 Vấn Đề Thực Tế

Tưởng tượng bạn cần xây dựng một Select component (dropdown). Có nhiều cách thiết kế API:

**Approach 1: Props-heavy API** ❌

```jsx
<Select
  options={[
    { value: '1', label: 'One', icon: '🥇' },
    { value: '2', label: 'Two', icon: '🥈' },
  ]}
  value={value}
  onChange={onChange}
  renderOption={(option) => (
    <div>
      {option.icon} {option.label}
    </div>
  )}
  renderTrigger={(value) => <button>{value}</button>}
/>
```

**Vấn đề:**

- API phức tạp, nhiều props
- Khó customize UI
- Render functions khó đọc
- Khó mở rộng khi thêm features

**Approach 2: Compound Components** ✅

```jsx
<Select
  value={value}
  onChange={onChange}
>
  <Select.Trigger>{value || 'Select...'}</Select.Trigger>
  <Select.Options>
    <Select.Option value='1'>🥇 One</Select.Option>
    <Select.Option value='2'>🥈 Two</Select.Option>
  </Select.Options>
</Select>
```

**Lợi ích:**

- API rõ ràng, dễ đọc
- Flexible - customize dễ dàng
- Compose được như HTML
- Mở rộng đơn giản

### 1.2 Giải Pháp - Compound Components Pattern

**Compound Components** là pattern cho phép các components "làm việc cùng nhau" để tạo thành một unit hoàn chỉnh, chia sẻ state ngầm định (implicit state) thông qua Context.

**Đặc điểm:**

1. **Parent component** quản lý shared state
2. **Child components** tự động access state qua Context
3. **Flexible composition** - user tự sắp xếp UI
4. **Implicit relationship** - không cần pass props manual

### 1.3 Mental Model

```
┌─────────────────────────────────────┐
│  Select (Parent)                    │
│  ┌─────────────────────────────┐    │
│  │ Context Provider            │    │
│  │ (value, onChange, isOpen)   │    │
│  └─────────────────────────────┘    │
│           │                         │
│           ├─ Select.Trigger ────────┼─> useContext(SelectContext)
│           │                         │   → auto access value, toggle
│           ├─ Select.Options ────────┼─> useContext(SelectContext)
│           │                         │   → auto access isOpen
│           └─ Select.Option ─────────┼─> useContext(SelectContext)
│                                     │   → auto access onChange
└─────────────────────────────────────┘

Giống như một "gia đình":
- Cha mẹ (Parent) giữ "tiền" (state)
- Con cái (Children) tự động "nhận tiền" khi cần (qua Context)
- Không cần cha mẹ đưa tận tay từng đứa (no props drilling)
```

### 1.4 Hiểu Lầm Phổ Biến

❌ **"Compound Components = nhiều components trong 1 file"**
→ Sai! Đó chỉ là cách tổ chức code. Compound Components là về **implicit state sharing** qua Context.

❌ **"Phải đặt tên kiểu Parent.Child"**
→ Không bắt buộc, nhưng là convention để code dễ đọc.

❌ **"Children phải render direct trong Parent"**
→ Không! Children có thể nested sâu, Context vẫn hoạt động.

❌ **"Compound Components là duy nhất cách tốt"**
→ Không! Mỗi pattern có trade-offs, phải biết khi nào dùng.

```jsx
 *
 * Features:
 * - Shared state qua Context

import { createContext, useContext, useState } from 'react';

// 1. Tạo Context
const ToggleContext = createContext(null);

// 2. Custom hook để access context
const useToggleContext = () => {
  const context = useContext(ToggleContext);
  if (!context) {
    throw new Error('Toggle components must be used within <Toggle>');
  }
  return context;
};

// 3. Parent Component
const Toggle = ({ children }) => {
  const [isOn, setIsOn] = useState(false);

  const toggle = () => setIsOn((prev) => !prev);

  return (
    <ToggleContext.Provider value={{ isOn, toggle }}>
      {children}
    </ToggleContext.Provider>
  );
};

// 4. Child Components
Toggle.Button = function ToggleButton({ children }) {
  const { toggle } = useToggleContext();

  return <button onClick={toggle}>{children}</button>;
};

Toggle.Status = function ToggleStatus() {
  const { isOn } = useToggleContext();

  return <span>Status: {isOn ? 'ON' : 'OFF'}</span>;
};

Toggle.Content = function ToggleContent({ children }) {
  const { isOn } = useToggleContext();

  return isOn ? <div>{children}</div> : null;
};

// ✅ CÁCH DÙNG - Flexible composition
function App() {
  return (
    <Toggle>
      <Toggle.Status />
      <Toggle.Button>Toggle Me</Toggle.Button>
      <Toggle.Content>
        <p>This content shows when ON</p>
      </Toggle.Content>
    </Toggle>
  );
}
```

**Tabs Component với validation và accessibility**

<details>
<summary>💡 Code Example</summary>

```jsx
/**
 * Tabs Component - Production-ready Compound Components
 *
 * Features:
 * - Keyboard navigation (Arrow keys)
 * - Accessibility (ARIA attributes)
 * - Controlled & Uncontrolled modes
 * - Error boundaries
 */

import {
  createContext,
  useContext,
  useState,
  useRef,
  useEffect,
  useCallback,
} from 'react';

/* =========================
   Context
========================= */

const TabsContext = createContext(null);

const useTabsContext = () => {
  const ctx = useContext(TabsContext);
  if (!ctx) {
    throw new Error('Tabs.* must be used within <Tabs>');
  }
  return ctx;
};

/* =========================
   Tabs Root (STATE OWNER)
========================= */

export function Tabs({
  children,
  value: controlledValue,
  defaultValue,
  onChange,
}) {
  const isControlled = controlledValue !== undefined;
  const [internalValue, setInternalValue] = useState(defaultValue ?? null);

  const value = isControlled ? controlledValue : internalValue;

  const setValue = useCallback(
    (v) => {
      if (!isControlled) setInternalValue(v);
      onChange?.(v);
    },
    [isControlled, onChange]
  );

  // 🔥 Registry (single source)
  const tabsRef = useRef([]);

  const registerTab = useCallback((node) => {
    if (node && !tabsRef.current.includes(node)) {
      tabsRef.current.push(node);
    }
  }, []);

  const unregisterTab = useCallback((node) => {
    tabsRef.current = tabsRef.current.filter((n) => n !== node);
  }, []);

  const focusTab = (index) => {
    const el = tabsRef.current[index];
    if (el) el.focus();
  };

  const contextValue = {
    value,
    setValue,
    tabsRef,
    registerTab,
    unregisterTab,
    focusTab,
  };

  return (
    <TabsContext.Provider value={contextValue}>
      {children}
    </TabsContext.Provider>
  );
}

/* =========================
   Tabs.List
========================= */

Tabs.List = function TabsList({ children }) {
  const { tabsRef, setValue } = useTabsContext();

  const handleKeyDown = (e) => {
    const tabs = tabsRef.current;
    const currentIndex = tabs.indexOf(e.target);

    if (currentIndex === -1) return;

    let nextIndex = null;

    switch (e.key) {
      case 'ArrowRight':
        nextIndex = (currentIndex + 1) % tabs.length;
        break;
      case 'ArrowLeft':
        nextIndex = (currentIndex - 1 + tabs.length) % tabs.length;
        break;
      case 'Home':
        nextIndex = 0;
        break;
      case 'End':
        nextIndex = tabs.length - 1;
        break;
      default:
        return;
    }

    e.preventDefault();

    const nextTab = tabs[nextIndex];
    nextTab?.focus();

    // 🔥 extract value từ dataset (avoid closure stale)
    const nextValue = nextTab?.dataset?.value;
    if (nextValue) setValue(nextValue);
  };

  return (
    <div role="tablist" onKeyDown={handleKeyDown}>
      {children}
    </div>
  );
};

/* =========================
   Tabs.Tab (ROVING TABINDEX)
========================= */

Tabs.Tab = function TabsTab({ value, children }) {
  const {
    value: selectedValue,
    setValue,
    registerTab,
    unregisterTab,
  } = useTabsContext();

  const ref = useRef(null);
  const isSelected = value === selectedValue;

  useEffect(() => {
    const node = ref.current;
    if (!node) return;

    registerTab(node);
    return () => unregisterTab(node);
  }, [registerTab, unregisterTab]);

  return (
    <button
      ref={ref}
      role="tab"
      data-value={value}
      id={`tab-${value}`}
      aria-selected={isSelected}
      aria-controls={`panel-${value}`}
      tabIndex={isSelected ? 0 : -1}
      onClick={() => setValue(value)}
      style={{
        padding: '8px 12px',
        cursor: 'pointer',
        borderBottom: isSelected
          ? '2px solid blue'
          : '2px solid transparent',
        fontWeight: isSelected ? 'bold' : 'normal',
      }}
    >
      {children}
    </button>
  );
};

/* =========================
   Tabs.Panel
========================= */

Tabs.Panel = function TabsPanel({ value, children }) {
  const { value: selectedValue } = useTabsContext();

  const isSelected = value === selectedValue;

  // 🔥 keep mounted (better UX + state preservation)
  return (
    <div
      role="tabpanel"
      id={`panel-${value}`}
      aria-labelledby={`tab-${value}`}
      hidden={!isSelected}
    >
      {isSelected && children}
    </div>
  );
};

// ✅ CÁCH DÙNG
function App() {
  const [activeTab, setActiveTab] = useState('profile');

  return (
    <div>
      {/* Controlled mode */}
      <Tabs
        value={activeTab}
        onChange={setActiveTab}
      >
        <Tabs.List>
          <Tabs.Tab value='profile'>Profile</Tabs.Tab>
          <Tabs.Tab value='settings'>Settings</Tabs.Tab>
          <Tabs.Tab value='billing'>Billing</Tabs.Tab>
        </Tabs.List>

        <Tabs.Panel value='profile'>
          <h2>Profile Content</h2>
          <p>User profile information...</p>
        </Tabs.Panel>

        <Tabs.Panel value='settings'>
          <h2>Settings Content</h2>
          <p>Application settings...</p>
        </Tabs.Panel>

        <Tabs.Panel value='billing'>
          <h2>Billing Content</h2>
          <p>Billing information...</p>
        </Tabs.Panel>
      </Tabs>

      {/* Uncontrolled mode */}
      <Tabs defaultValue='home'>
        <Tabs.List>
          <Tabs.Tab value='home'>Home</Tabs.Tab>
          <Tabs.Tab value='about'>About</Tabs.Tab>
        </Tabs.List>
        <Tabs.Panel value='home'>Home content</Tabs.Panel>
        <Tabs.Panel value='about'>About content</Tabs.Panel>
      </Tabs>
    </div>
  );
}

/*
Kết quả:
- ✅ Keyboard navigation works (Arrow keys, Home, End)
- ✅ Screen reader accessible
- ✅ Both controlled/uncontrolled modes
- ✅ No props drilling
*/
```

**Accordion với nested context và performance optimization**

<details>
<summary>💡 Code Example</summary>

```jsx
/**
 * Accordion Component - Edge Cases Handling
 *
 * Edge Cases:
 * - Multiple accordions on page (isolated state)
 * - Nested accordions (context conflicts)
 * - Dynamic items (add/remove)
 * - Performance with many items
 */

import {
  createContext,
  useContext,
  useState,
  useCallback,
  useMemo,
} from 'react';

// ==================== Contexts ====================

const AccordionContext = createContext(null);
const ItemContext = createContext(null);

const useAccordionContext = () => {
  const context = useContext(AccordionContext);
  if (!context) {
    throw new Error('Accordion sub-components must be used within <Accordion>');
  }
  return context;
};

const useItemContext = () => {
  const context = useContext(ItemContext);
  if (!context) {
    throw new Error('Accordion.Trigger and Accordion.Content must be used within <Accordion.Item>');
  }
  return context;
};

// ==================== Components ====================

const Accordion = ({ children, allowMultiple = false }) => {
  const [openItems, setOpenItems] = useState(new Set());

  const toggleItem = useCallback(
    (id) => {
      setOpenItems((prev) => {
        const next = new Set(prev);

        if (next.has(id)) {
          next.delete(id);
        } else {
          if (!allowMultiple) {
            next.clear();
          }
          next.add(id);
        }

        return next;
      });
    },
    [allowMultiple],
  );

  const isItemOpen = useCallback(
    (id) => openItems.has(id),
    [openItems],
  );

  const value = useMemo(
    () => ({
      toggleItem,
      isItemOpen,
    }),
    [toggleItem, isItemOpen],
  );

  return (
    <AccordionContext.Provider value={value}>
      <div className="accordion">{children}</div>
    </AccordionContext.Provider>
  );
};

Accordion.Item = function AccordionItem({ id, children }) {
  const { isItemOpen } = useAccordionContext();
  const open = isItemOpen(id);

  const itemValue = useMemo(() => ({ id, open }), [id, open]);

  return (
    <ItemContext.Provider value={itemValue}>
      <div className="accordion-item" data-state={open ? 'open' : 'closed'}>
        {children}
      </div>
    </ItemContext.Provider>
  );
};

Accordion.Trigger = function AccordionTrigger({ children, ...props }) {
  const { id, open } = useItemContext();
  const { toggleItem } = useAccordionContext();

  return (
    <button
      {...props}
      onClick={() => toggleItem(id)}
      aria-expanded={open}
      className="accordion-trigger"
    >
      {children}
    </button>
  );
};

Accordion.Content = function AccordionContent({ children, ...props }) {
  const { id, open } = useItemContext();

  if (!open) return null;

  return (
    <div {...props} className="accordion-content">
      {children}
    </div>
  );
};

// ==================== Usage ====================

// 1. Single mode (default)
function SingleAccordion() {
  return (
    <Accordion>
      <Accordion.Item id="item1">
        <Accordion.Trigger>Item 1</Accordion.Trigger>
        <Accordion.Content>Content 1</Accordion.Content>
      </Accordion.Item>

      <Accordion.Item id="item2">
        <Accordion.Trigger>Item 2</Accordion.Trigger>
        <Accordion.Content>Content 2</Accordion.Content>
      </Accordion.Item>
    </Accordion>
  );
}

// 2. Multiple mode
function MultipleAccordion() {
  return (
    <Accordion allowMultiple>
      <Accordion.Item id="a">
        <Accordion.Trigger>Section A</Accordion.Trigger>
        <Accordion.Content>Content A</Accordion.Content>
      </Accordion.Item>
      <Accordion.Item id="b">
        <Accordion.Trigger>Section B</Accordion.Trigger>
        <Accordion.Content>Content B</Accordion.Content>
      </Accordion.Item>
    </Accordion>
  );
}

// 3. Nested accordions
function NestedAccordion() {
  return (
    <Accordion>
      <Accordion.Item id="parent1">
        <Accordion.Trigger>Parent 1</Accordion.Trigger>
        <Accordion.Content>
          <Accordion>
            <Accordion.Item id="child1">
              <Accordion.Trigger>Child 1</Accordion.Trigger>
              <Accordion.Content>Nested content</Accordion.Content>
            </Accordion.Item>
          </Accordion>
        </Accordion.Content>
      </Accordion.Item>
    </Accordion>
  );
}

// 4. Dynamic items
function DynamicAccordion() {
  const [items, setItems] = useState([
    { id: '1', title: 'Item 1', content: 'Content 1' },
    { id: '2', title: 'Item 2', content: 'Content 2' },
  ]);

  const addItem = () => {
    const id = String(Date.now());
    setItems((prev) => [
      ...prev,
      { id, title: `Item ${prev.length + 1}`, content: `Content ${prev.length + 1}` },
    ]);
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <Accordion>
        {items.map((item) => (
          <Accordion.Item key={item.id} id={item.id}>
            <Accordion.Trigger>{item.title}</Accordion.Trigger>
            <Accordion.Content>{item.content}</Accordion.Content>
          </Accordion.Item>
        ))}
      </Accordion>
    </div>
  );
}

export { Accordion, SingleAccordion, MultipleAccordion, NestedAccordion, DynamicAccordion };

/*
Edge Cases Handled:
✅ Multiple instances isolated (mỗi <Accordion> có state riêng)
✅ Nested accordions không conflict context
✅ Dynamic items (add/remove) works
✅ Performance optimized (useMemo, useCallback)
✅ allowMultiple mode
*/
```

```jsx
/**
 * Counter Component - Compound Components Pattern
 *
 * @example
 * <Counter>
 *   <Counter.Display />
 *   <Counter.Increment />
 *   <Counter.Decrement />
 *   <Counter.Reset />
 * </Counter>
 */

import { createContext, useContext, useState } from 'react';

// 1. Tạo Context
const CounterContext = createContext(null);

// 2. Custom hook để access context
const useCounterContext = () => {
  const context = useContext(CounterContext);
  if (!context) {
    throw new Error('Counter.* components must be used within <Counter>');
  }
  return context;
};

// 3. Parent Component
const Counter = ({ children, initialValue = 0 }) => {
  const [count, setCount] = useState(initialValue);

  const increment = () => setCount((prev) => prev + 1);
  const decrement = () => setCount((prev) => prev - 1);
  const reset = () => setCount(initialValue);

  return (
    <CounterContext.Provider value={{ count, increment, decrement, reset }}>
      <div className='counter'>{children}</div>
    </CounterContext.Provider>
  );
};

// 4. Child Components
Counter.Display = function CounterDisplay() {
  const { count } = useCounterContext();
  return <div className='count'>Count: {count}</div>;
};

Counter.Increment = function CounterIncrement({ children = '+1' }) {
  const { increment } = useCounterContext();
  return <button onClick={increment}>{children}</button>;
};

Counter.Decrement = function CounterDecrement({ children = '-1' }) {
  const { decrement } = useCounterContext();
  return <button onClick={decrement}>{children}</button>;
};

Counter.Reset = function CounterReset({ children = 'Reset' }) {
  const { reset } = useCounterContext();
  return <button onClick={reset}>{children}</button>;
};

// Test
function App() {
  return (
    <div>
      {/* Flexible composition */}
      <Counter initialValue={10}>
        <Counter.Display />
        <div style={{ display: 'flex', gap: '8px' }}>
          <Counter.Increment>➕</Counter.Increment>
          <Counter.Decrement>➖</Counter.Decrement>
          <Counter.Reset>🔄</Counter.Reset>
        </div>
      </Counter>

      {/* Khác layout */}
      <Counter>
        <div className='card'>
          <h3>My Counter</h3>
          <Counter.Display />
          <Counter.Increment />
          <Counter.Decrement />
        </div>
        <Counter.Reset />
      </Counter>
    </div>
  );
}

Kết quả:
```

```jsx
const Modal = ({ children, open, onOpenChange }) => {
  const [internalOpen, setInternalOpen] = useState(false);
  const isControlled = open !== undefined;
  const isOpen = isControlled ? open : internalOpen;

  const setIsOpen = useCallback(
    (next) => {
      if (!isControlled) setInternalOpen(next);
      onOpenChange?.(next);
    },
    [isControlled, onOpenChange],
  );

  const close = useCallback(() => setIsOpen(false), [setIsOpen]);

  return (
    <ModalContext.Provider value={{ isOpen, setIsOpen, close }}>

    </ModalContext.Provider>

Modal.Content = function ModalContent({ children }) {
  const { isOpen, close } = useModalContext();
  const modalRef = useRef(null);
  const previousActiveElement = useRef(null);

  // ESC + Focus Trap
  useEffect(() => {
    if (!isOpen) return;

    const getFocusable = () => {
      return Array.from(
        modalRef.current?.querySelectorAll(
          'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
        ) || []
      ).filter((el) => !el.disabled && el.offsetParent !== null);
    };

    const handleKeyDown = (e) => {
      if (e.key === 'Escape') {
        close();
        return;
      }

      if (e.key !== 'Tab') return;

      const focusable = getFocusable();
      if (focusable.length === 0) {
        e.preventDefault();
        return;
      }

      const first = focusable[0];
      const last = focusable[focusable.length - 1];

      if (e.shiftKey) {
        if (document.activeElement === first) {
          e.preventDefault();
          last.focus();
        }
      } else {
        if (document.activeElement === last) {
          e.preventDefault();
          first.focus();
        }
      }
    };

    document.addEventListener('keydown', handleKeyDown);
    return () => document.removeEventListener('keydown', handleKeyDown);
  }, [isOpen, close]);

  // Lock body scroll
  useEffect(() => {
    if (!isOpen) return;
    const original = document.body.style.overflow;
    document.body.style.overflow = 'hidden';
    return () => {
      document.body.style.overflow = original;
    };
  }, [isOpen]);

  // Focus first element + restore
  useEffect(() => {
    if (!isOpen) return;

    previousActiveElement.current = document.activeElement;

    const focusable = modalRef.current?.querySelectorAll(
      'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
    );
    focusable?.[0]?.focus();

    return () => {
      previousActiveElement.current?.focus();
    };
  }, [isOpen]);

  // Click outside
  const handleOverlayClick = (e) => {
    if (e.target === e.currentTarget) close();
  };

  if (!isOpen) return null;

  return createPortal(
    <div
      className="modal-overlay"
      onClick={handleOverlayClick}
      style={{
        position: 'fixed',
        inset: 0,
        backgroundColor: 'rgba(0,0,0,0.5)',
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
        zIndex: 1000,
      }}
    >
      <div
        ref={modalRef}
        role="dialog"
        aria-modal="true"
        className="modal-content"
        style={{
          background: 'white',
          borderRadius: 8,
          padding: 24,
          maxWidth: 500,
          width: '90%',
          maxHeight: '80vh',
          overflow: 'auto',
        }}
      >
        {children}
      </div>
    </div>,
    document.body
  );
};
```

```jsx
 * 2. Composability: Dễ thêm custom fields
 * 3. Clear API: Readable như HTML forms
 * 4. Validation: Có thể attach validation per field
 * 5. Extensibility: Dễ thêm Form.Checkbox, Form.Select, etc.
 *
 * Consequences:
 * ✅ Pros:
 * - Rất flexible và extensible
 * - API rõ ràng, dễ maintain

 * - Dễ thêm field types mới

 * - Nhiều code hơn khi sử dụng
 * - Dễ sai structure (quên closing tags)
 * - Performance: nhiều components render
 *
 * Alternatives Considered:
 * - Props-based: Đơn giản nhưng không flexible
 * - Hybrid: Phức tạp, không rõ ràng
 *

import { createContext, useContext, useState, useCallback } from 'react';

const FormContext = createContext(null);

const useFormContext = () => {
  const context = useContext(FormContext);
  if (!context) {
    throw new Error('Form.* components must be used within <Form>');
  }
  return context;
};

const Form = ({ children, onSubmit, initialValues = {} }) => {
  const [values, setValues] = useState(initialValues);
  const [errors, setErrors] = useState({});
  const [touched, setTouched] = useState({});

  const setValue = useCallback((name, value) => {
    setValues((prev) => ({ ...prev, [name]: value }));
  }, []);

  const setError = useCallback((name, error) => {
    setErrors((prev) => ({ ...prev, [name]: error }));
  }, []);

  const setFieldTouched = useCallback((name) => {
    setTouched((prev) => ({ ...prev, [name]: true }));
  }, []);

  const reset = useCallback(() => {
    setValues(initialValues);
    setErrors({});
    setTouched({});
  }, [initialValues]);

  const handleSubmit = (e) => {
    e.preventDefault();

    // Mark all fields as touched
    const allFieldsTouched = Object.keys(values).reduce((acc, key) => {
      acc[key] = true;
      return acc;
    }, {});
    setTouched(allFieldsTouched);

    // Check if any errors
    const hasErrors = Object.values(errors).some((error) => error);

    if (!hasErrors) {
      onSubmit?.(values);
    }
  };

  return (
    <FormContext.Provider
      value={{
        values,
        errors,
        touched,
        setValue,
        setError,
        setFieldTouched,
        reset,
      }}
    >
      <form onSubmit={handleSubmit}>{children}</form>
    </FormContext.Provider>
  );
};

  const error = errors[name];
  const isTouched = touched[name];

  const handleChange = (e) => {
    const newValue = e.target.value;
    setValue(name, newValue)
    if(error) setError(name,'');

    setFieldTouched(name);
    // Validate

        style={{ display: 'block', marginBottom: '4px' }}

        {label}

        name={name}
        type={type}
```

```jsx
/**
 * Menu Component System - Production Ready
 *
 * Features:
 * ✅ Multi-level nested menus
 * ✅ Full keyboard navigation
 * ✅ Click outside to close
 * ✅ Hover behavior
 * ✅ Disabled items

 *   <Menu.Trigger>File</Menu.Trigger>

import {
  createContext,
  useContext,
  useState,
  useRef,
  useEffect,
  useLayoutEffect,
  useCallback,
  useId,
  useMemo,
} from 'react';
import { createPortal } from 'react-dom';

// ==================== Contexts ====================

const MenuContext = createContext(null);

const useMenuContext = () => {
  const ctx = useContext(MenuContext);
  if (!ctx) throw new Error('Menu.* must be used within <Menu>');
  return ctx;
};

// ==================== Helpers ====================

const isPrintableKey = (e) => {
  return e.key.length === 1 && !e.ctrlKey && !e.metaKey && !e.altKey;
};

// ==================== Menu Root ====================

const Menu = ({ children, open, onOpenChange }) => {
  const [internalOpen, setInternalOpen] = useState(false);
  const isControlled = open !== undefined;
  const isOpen = isControlled ? open : internalOpen;

  const menuId = useId();
  const triggerRef = useRef(null);
  const contentRef = useRef(null);

  const setIsOpen = useCallback(
    (next) => {
      if (!isControlled) setInternalOpen(next);
      onOpenChange?.(next);
    },
    [isControlled, onOpenChange],
  );

  const close = useCallback(() => setIsOpen(false), [setIsOpen]);

  const itemsRef = useRef([]);
  const registerItem = useCallback((item) => {
    itemsRef.current.push(item);
    return () => {
      itemsRef.current = itemsRef.current.filter((i) => i.id !== item.id);
    };
  }, []);

  const getEnabledItems = useCallback(() => {
    return itemsRef.current.filter((i) => !i.disabled);
  }, []);

  const focusItem = useCallback((index) => {
    const items = getEnabledItems();
    const target = items[index];
    if (target?.ref.current) {
      target.ref.current.focus();
      target.ref.current.scrollIntoView({ block: 'nearest' });
    }
  }, [getEnabledItems]);

  const focusFirst = useCallback(() => focusItem(0), [focusItem]);
  const focusLast = useCallback(() => {
    const items = getEnabledItems();
    focusItem(items.length - 1);
  }, [focusItem, getEnabledItems]);

  const focusNext = useCallback(() => {
    const items = getEnabledItems();
    const active = document.activeElement;
    const currentIndex = items.findIndex((i) => i.ref.current === active);
    const next = currentIndex >= 0 ? (currentIndex + 1) % items.length : 0;
    focusItem(next);
  }, [focusItem, getEnabledItems]);

  const focusPrev = useCallback(() => {
    const items = getEnabledItems();
    const active = document.activeElement;
    const currentIndex = items.findIndex((i) => i.ref.current === active);
    const prev = currentIndex >= 0
      ? (currentIndex - 1 + items.length) % items.length
      : items.length - 1;
    focusItem(prev);
  }, [focusItem, getEnabledItems]);

  const value = useMemo(
    () => ({
      isOpen,
      setIsOpen,
      close,
      menuId,
      triggerRef,
      contentRef,
      registerItem,
      focusFirst,
      focusLast,
      focusNext,
      focusPrev,
      getEnabledItems,
    }),
    [
      isOpen, setIsOpen, close, menuId,
      registerItem, focusFirst, focusLast, focusNext, focusPrev, getEnabledItems,
    ],
  );

  return (
    <MenuContext.Provider value={value}>
      <div style={{ position: 'relative', display: 'inline-block' }}>
        {children}
      </div>
    </MenuContext.Provider>
  );
};

// ==================== Trigger ====================

Menu.Trigger = function MenuTrigger({ children, asChild }) {
  const { isOpen, setIsOpen, menuId, triggerRef } = useMenuContext();

  const props = {
    ref: triggerRef,
    id: `${menuId}-trigger`,
    'aria-haspopup': 'menu',
    'aria-expanded': isOpen,
    'aria-controls': `${menuId}-content`,
    onClick: () => setIsOpen(!isOpen),
    onKeyDown: (e) => {
      if (e.key === 'ArrowDown') {
        e.preventDefault();
        setIsOpen(true);
      }
    },
  };

  if (asChild) {
    return children(props);
  }

  return (
    <button
      {...props}
      style={{
        padding: '8px 16px',
        cursor: 'pointer',
        border: '1px solid #ccc',
        borderRadius: 4,
        background: 'white',
      }}
    >
      {children}
    </button>
  );
};

// ==================== Content ====================

Menu.Content = function MenuContent({ children, align = 'start' }) {
  const {
    isOpen,
    close,
    menuId,
    triggerRef,
    contentRef,
    focusFirst,
    focusNext,
    focusPrev,
    focusLast,
    getEnabledItems,
  } = useMenuContext();

  const [position, setPosition] = useState({ top: 0, left: 0 });

  useLayoutEffect(() => {
    if (!isOpen || !triggerRef.current) return;
    const rect = triggerRef.current.getBoundingClientRect();
    setPosition({
      top: rect.bottom + window.scrollY + 4,
      left: align === 'start'
        ? rect.left + window.scrollX
        : rect.right + window.scrollX - 200,
    });
  }, [isOpen, align, triggerRef]);

  useEffect(() => {
    if (!isOpen) return;
    const handlePointerDown = (e) => {
      const target = e.target;
      const content = contentRef.current;
      const trigger = triggerRef.current;
      if (
        content &&
        !content.contains(target) &&
        trigger &&
        !trigger.contains(target)
      ) {
        close();
      }
    };
    document.addEventListener('pointerdown', handlePointerDown);
    return () => document.removeEventListener('pointerdown', handlePointerDown);
  }, [isOpen, close, contentRef, triggerRef]);

  useEffect(() => {
    if (!isOpen) return;
    const handleKeyDown = (e) => {
      switch (e.key) {
        case 'ArrowDown':
          e.preventDefault();
          focusNext();
          break;
        case 'ArrowUp':
          e.preventDefault();
          focusPrev();
          break;
        case 'Home':
          e.preventDefault();
          focusFirst();
          break;
        case 'End':
          e.preventDefault();
          focusLast();
          break;
        case 'Escape':
          e.preventDefault();
          close();
          triggerRef.current?.focus();
          break;
        case 'Tab':
          e.preventDefault();
          focusNext();
          break;
        default:
          if (isPrintableKey(e)) {
            const items = getEnabledItems();
            const chars = e.key.toLowerCase();
            const active = document.activeElement;
            const currentIdx = items.findIndex((i) => i.ref.current === active);
            const next = items.findIndex((i, idx) => {
              if (idx <= currentIdx) return false;
              const text = i.ref.current?.textContent?.toLowerCase() || '';
              return text.startsWith(chars);
            });
            if (next !== -1) {
              items[next].ref.current?.focus();
            } else {
              const wrap = items.findIndex((i) => {
                const text = i.ref.current?.textContent?.toLowerCase() || '';
                return text.startsWith(chars);
              });
              if (wrap !== -1) items[wrap].ref.current?.focus();
            }
          }
      }
    };
    document.addEventListener('keydown', handleKeyDown);
    return () => document.removeEventListener('keydown', handleKeyDown);
  }, [isOpen, close, focusNext, focusPrev, focusFirst, focusLast, triggerRef, getEnabledItems]);

  useEffect(() => {
    if (isOpen) {
      requestAnimationFrame(() => focusFirst());
    }
  }, [isOpen, focusFirst]);

  if (!isOpen) return null;

  return createPortal(
    <div
      ref={contentRef}
      id={`${menuId}-content`}
      role="menu"
      aria-labelledby={`${menuId}-trigger`}
      style={{
        position: 'absolute',
        top: position.top,
        left: position.left,
        backgroundColor: 'white',
        border: '1px solid #ccc',
        borderRadius: 4,
        boxShadow: '0 4px 12px rgba(0,0,0,0.15)',
        minWidth: 200,
        padding: '4px 0',
        zIndex: 1000,
        outline: 'none',
      }}
    >
      {children}
    </div>,
    document.body,
  );
};

// ==================== Menu Item ====================

Menu.Item = function MenuItem({ children, onSelect, disabled = false, icon }) {
  const { close, registerItem, triggerRef } = useMenuContext();
  const itemId = useId();
  const ref = useRef(null);
  const [isFocused, setIsFocused] = useState(false);

  useEffect(() => {
    return registerItem({ id: itemId, ref, disabled });
  }, [itemId, disabled, registerItem]);

  const handleClick = () => {
    if (disabled) return;
    onSelect?.();
    close();
    triggerRef.current?.focus();
  };

  const handleKeyDown = (e) => {
    if (disabled) return;
    if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault();
      handleClick();
    }
  };

  const bg = disabled
    ? 'transparent'
    : isFocused
      ? '#f0f0f0'
      : 'transparent';

  return (
    <button
      ref={ref}
      role="menuitem"
      tabIndex={-1}
      disabled={disabled}
      onClick={handleClick}
      onKeyDown={handleKeyDown}
      onFocus={() => setIsFocused(true)}
      onBlur={() => setIsFocused(false)}
      onMouseEnter={(e) => {
        if (!disabled) {
          e.currentTarget.focus();
        }
      }}
      style={{
        display: 'flex',
        alignItems: 'center',
        gap: 8,
        width: '100%',
        padding: '8px 16px',
        textAlign: 'left',
        border: 'none',
        background: bg,
        cursor: disabled ? 'not-allowed' : 'pointer',
        opacity: disabled ? 0.5 : 1,
        outline: 'none',
        transition: 'background-color 0.1s ease',
      }}
    >
      {icon && <span aria-hidden>{icon}</span>}
      {children}
    </button>
  );
};

Menu.Submenu = function MenuSubmenu({ label, children, icon, disabled = false }) {
  const [isOpen, setIsOpen] = useState(false);
  const [isFocused, setIsFocused] = useState(false);
  const { registerItem, close: closeParent } = useMenuContext();

    return registerItem({ id: itemId, ref: buttonRef, disabled });
  }, [itemId, disabled, registerItem]);

  const openSubmenu = useCallback(() => {
    clearTimeout(timeoutRef.current);
    setIsOpen(true);
  }, []);

  const closeSubmenu = useCallback(() => {
    timeoutRef.current = setTimeout(() => setIsOpen(false), 150);
  }, []);

  const handleMouseEnter = useCallback(() => {
    clearTimeout(timeoutRef.current);
    setIsOpen(true);
  }, []);

  const handleMouseLeave = useCallback(() => {
    timeoutRef.current = setTimeout(() => setIsOpen(false), 150);
  }, []);

  const handleKeyDown = (e) => {
    if (disabled) return;
    if (e.key === 'ArrowRight') {
      e.preventDefault();
      openSubmenu();
      // Focus first item after submenu renders
      setTimeout(() => {
        const first = submenuRef.current?.querySelector('[role="menuitem"]');
        first?.focus();
      }, 0);
    } else if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault();
      openSubmenu();
      setTimeout(() => {
        const first = submenuRef.current?.querySelector('[role="menuitem"]');
        first?.focus();
      }, 0);
    }
  };

  // Position submenu
  const [subPosition, setSubPosition] = useState({ top: 0, left: 0 });
  useLayoutEffect(() => {
    if (!isOpen || !buttonRef.current) return;

      top: rect.top + window.scrollY,
      left: rect.right + window.scrollX + 2,
    });

  // Click outside submenu
  useEffect(() => {
    if (!isOpen) return;
    const handlePointerDown = (e) => {
      if (
        submenuRef.current &&
        !submenuRef.current.contains(e.target) &&
        buttonRef.current &&
        !buttonRef.current.contains(e.target)

      }
    };
    document.addEventListener('pointerdown', handlePointerDown);
    return () => document.removeEventListener('pointerdown', handlePointerDown);
  }, [isOpen]);

  const bg = disabled
    ? 'transparent'
    : isOpen || isFocused
      ? '#f0f0f0'
      : 'transparent';

    <div
      role="none"
      onMouseEnter={handleMouseEnter}
      onMouseLeave={handleMouseLeave}
      style={{ position: 'relative' }}
    >
      <button
        ref={buttonRef}
        role="menuitem"
        aria-haspopup="menu"
        aria-expanded={isOpen}
        tabIndex={-1}
        disabled={disabled}
        onKeyDown={handleKeyDown}
        onFocus={() => setIsFocused(true)}
        onBlur={() => setIsFocused(false)}
        style={{
          display: 'flex',
          alignItems: 'center',
          justifyContent: 'space-between',

          {label}

        <span aria-hidden>▶</span>
      </button>

      {isOpen &&
        createPortal(

            ref={submenuRef}
            role="menu"
            style={{
              position: 'absolute',
              top: subPosition.top,
              left: subPosition.left,

                buttonRef.current?.focus();
```

| **Readability**    | ⭐⭐⭐⭐        | ⭐⭐⭐⭐⭐          | ⭐⭐         | ⭐⭐⭐   |
| **Learning Curve** | ⭐⭐⭐⭐⭐      | ⭐⭐⭐⭐            | ⭐⭐         | ⭐⭐     |

### Chi tiết từng approach:

**1. Props-based API**

```jsx
<Select
  options={options}
  value={value}
  onChange={onChange}
  renderOption={(opt) => <div>{opt.label}</div>}
/>
```

- ✅ Đơn giản khi dùng
- ✅ Type-safe với TypeScript
- ✅ Ít code hơn
- ❌ Khó customize phức tạp
- ❌ Render props khó đọc
- **Khi nào dùng:** Simple components, fixed UI structure

**2. Compound Components** ⭐ Recommended

```jsx
<Select
  value={value}
  onChange={onChange}
>
  <Select.Trigger>Select...</Select.Trigger>
  <Select.Options>
    <Select.Option value='1'>One</Select.Option>
  </Select.Options>
</Select>
```

- ✅ Rất flexible
- ✅ Readable như HTML
- ✅ Dễ customize
- ✅ Implicit state sharing
- ❌ Nhiều code hơn
- ❌ Dễ sai structure
- **Khi nào dùng:** Complex UI, need flexibility, reusable libraries

**3. Render Props** (Legacy - hiểu để đọc code cũ)

```jsx
<Select>
  {({ isOpen, value }) => (
    <div>
      {isOpen ? 'Open' : 'Closed'}: {value}
    </div>
  )}
</Select>
```

- ✅ Flexible logic
- ✅ Explicit state
- ❌ Callback hell
- ❌ Khó đọc
- **Khi nào dùng:** Không - dùng Hooks thay thế

**4. HOCs** (Legacy - hiểu để đọc code cũ)

```jsx
const EnhancedSelect = withDropdown(Select);
```

- ✅ Reuse logic
- ❌ Props collision
- ❌ Wrapper hell
- ❌ Static composition
- **Khi nào dùng:** Không - dùng Hooks thay thế

### Decision Tree

```
Bạn cần component có API như thế nào?

├─ Simple, fixed structure?
│  └─> Props-based API
│     Example: <Button variant="primary" />
│
├─ Complex, customizable UI?
│  ├─ Component library?
│  │  └─> Compound Components
│  │     Example: <Tabs>, <Menu>, <Accordion>
│  │
│  └─ One-off component?
│     └─> Props-based API (đơn giản hơn)
│
├─ Share logic giữa components?
│  └─> Custom Hooks
│     Example: useFetch, useToggle
│
└─ Đọc legacy code?
   ├─ Thấy render props?
   │  └─> Hiểu pattern, cân nhắc refactor
   │
   └─ Thấy HOCs?
      └─> Hiểu pattern, cân nhắc refactor

RECOMMENDATION:
- Default: Compound Components cho complex components
- Fallback: Props-based nếu quá đơn giản
- Never: Render Props, HOCs (except legacy code)
```

```jsx
import { useMemo, useCallback } from 'react';

const Form = ({ children }) => {
  const [values, setValues] = useState({});
  const [errors, setErrors] = useState({});

  // Memoize functions
  const setValue = useCallback((name, value) => {
    setValues((prev) => ({ ...prev, [name]: value }));
  }, []);

  const setError = useCallback((name, error) => {
    setErrors((prev) => ({ ...prev, [name]: error }));
  }, []);

  // Memoize context value
  const value = useMemo(
    () => ({
      values,
      errors,
      setValue,
      setError,
    }),
    [values, errors, setValue, setError],
  );

  return <FormContext.Provider value={value}>{children}</FormContext.Provider>;
};
```

```jsx
// Context riêng cho state và dispatch
const FormStateContext = createContext(null);
const FormDispatchContext = createContext(null);

const Form = ({ children }) => {
  const [values, setValues] = useState({});
  const [errors, setErrors] = useState({});

  const setValue = useCallback((name, value) => {
    setValues((prev) => ({ ...prev, [name]: value }));
  }, []);

  const setError = useCallback((name, error) => {
    setErrors((prev) => ({ ...prev, [name]: error }));
  }, []);

  // State context (thay đổi khi values/errors change)
  const stateValue = useMemo(
    () => ({
      values,
      errors,
    }),
    [values, errors],
  );

  // Dispatch context (KHÔNG BAO GIỜ thay đổi)
  const dispatchValue = useMemo(
    () => ({
      setValue,
      setError,
    }),
    [setValue, setError],

    <FormDispatchContext.Provider value={dispatchValue}>
      <FormStateContext.Provider value={stateValue}>
        {children}
      </FormStateContext.Provider>
```

```jsx
import {
  createContext,
  useContext,
  useState,
  useRef,
  useEffect,
  useLayoutEffect,
  useCallback,
  useId,
  useMemo,
} from 'react';
import { createPortal } from 'react-dom';

// ==================== Contexts ====================

const MenuContext = createContext(null);

const useMenuContext = () => {
  const ctx = useContext(MenuContext);
  if (!ctx) throw new Error('Menu.* must be used within <Menu>');
  return ctx;
};

// ==================== Helpers ====================

const isPrintableKey = (e) => {
  return e.key.length === 1 && !e.ctrlKey && !e.metaKey && !e.altKey;
};

const normalizeText = (text) => text?.toLowerCase().trim() || '';

// ==================== Menu Root ====================

const Menu = ({ children, open, onOpenChange }) => {
  const [internalOpen, setInternalOpen] = useState(false);
  const isControlled = open !== undefined;
  const isOpen = isControlled ? open : internalOpen;

  const menuId = useId();
  const triggerRef = useRef(null);
  const contentRef = useRef(null);
  const searchRef = useRef(null);

  const [searchQuery, setSearchQuery] = useState('');

  const setIsOpen = useCallback(
    (next) => {
      if (!isControlled) setInternalOpen(next);
      onOpenChange?.(next);
      if (!next) setSearchQuery('');
    },
    [isControlled, onOpenChange],
  );

  const close = useCallback(() => setIsOpen(false), [setIsOpen]);

  const itemsRef = useRef([]);
  const registerItem = useCallback((item) => {
    itemsRef.current.push(item);
    return () => {
      itemsRef.current = itemsRef.current.filter((i) => i.id !== item.id);
    };
  }, []);

  const getEnabledItems = useCallback(() => {
    return itemsRef.current.filter((i) => !i.disabled && !i.hidden);
  }, []);

  const focusItem = useCallback((index) => {
    const items = getEnabledItems();
    const target = items[index];
    if (target?.ref.current) {
      target.ref.current.focus();
      target.ref.current.scrollIntoView({ block: 'nearest' });
    }
  }, [getEnabledItems]);

  const focusFirst = useCallback(() => focusItem(0), [focusItem]);
  const focusLast = useCallback(() => {
    const items = getEnabledItems();
    focusItem(items.length - 1);
  }, [focusItem, getEnabledItems]);

  const focusNext = useCallback(() => {
    const items = getEnabledItems();
    const active = document.activeElement;
    const currentIndex = items.findIndex((i) => i.ref.current === active);
    const next = currentIndex >= 0 ? (currentIndex + 1) % items.length : 0;
    focusItem(next);
  }, [focusItem, getEnabledItems]);

  const focusPrev = useCallback(() => {
    const items = getEnabledItems();
    const active = document.activeElement;
    const currentIndex = items.findIndex((i) => i.ref.current === active);
    const prev = currentIndex >= 0
      ? (currentIndex - 1 + items.length) % items.length
      : items.length - 1;
    focusItem(prev);
  }, [focusItem, getEnabledItems]);

  const value = useMemo(
    () => ({
      isOpen,
      setIsOpen,
      close,
      menuId,
      triggerRef,
      contentRef,
      searchRef,
      searchQuery,
      setSearchQuery,
      registerItem,
      focusFirst,
      focusLast,
      focusNext,
      focusPrev,
      getEnabledItems,
    }),
    [
      isOpen, setIsOpen, close, menuId,
      searchQuery, setSearchQuery,
      registerItem, focusFirst, focusLast, focusNext, focusPrev, getEnabledItems,
    ],
  );

  return (
    <MenuContext.Provider value={value}>
      <div style={{ position: 'relative', display: 'inline-block' }}>
        {children}
      </div>
    </MenuContext.Provider>
  );
};

// ==================== Content ====================

Menu.Content = function MenuContent({ children, align = 'start' }) {
  const {
    isOpen,
    close,
    menuId,
    triggerRef,
    contentRef,
    focusFirst,
    focusNext,
    focusPrev,
    focusLast,
    getEnabledItems,
    searchRef,
    searchQuery,
  } = useMenuContext();

  const [position, setPosition] = useState({ top: 0, left: 0 });

  useLayoutEffect(() => {
    if (!isOpen || !triggerRef.current) return;
    const rect = triggerRef.current.getBoundingClientRect();
    setPosition({
      top: rect.bottom + window.scrollY + 4,
      left: align === 'start'
        ? rect.left + window.scrollX
        : rect.right + window.scrollX - 200,
    });
  }, [isOpen, align, triggerRef]);

  useEffect(() => {
    if (!isOpen) return;
    const handlePointerDown = (e) => {
      const target = e.target;
      const content = contentRef.current;
      const trigger = triggerRef.current;
      if (
        content &&
        !content.contains(target) &&
        trigger &&
        !trigger.contains(target)
      ) {
        close();
      }
    };
    document.addEventListener('pointerdown', handlePointerDown);
    return () => document.removeEventListener('pointerdown', handlePointerDown);
  }, [isOpen, close, contentRef, triggerRef]);

  // Refocus when search filters change and current focus is lost
  useEffect(() => {
    if (!isOpen || !searchQuery) return;
    const active = document.activeElement;
    const items = getEnabledItems();
    const stillVisible = items.some((i) => i.ref.current === active);
    if (!stillVisible && items.length > 0) {
      requestAnimationFrame(() => focusFirst());
    }
  }, [searchQuery, isOpen, getEnabledItems, focusFirst]);

  useEffect(() => {
    if (!isOpen) return;
    const handleKeyDown = (e) => {
      const active = document.activeElement;

      if (active === searchRef.current) {
        if (e.key === 'ArrowDown') {
          e.preventDefault();
          focusFirst();
          return;
        }
        if (e.key === 'Escape') {
          e.preventDefault();
          close();
          triggerRef.current?.focus();
          return;
        }
        return;
      }

      switch (e.key) {
        case 'ArrowDown':
          e.preventDefault();
          focusNext();
          break;
        case 'ArrowUp':
          e.preventDefault();
          focusPrev();
          break;
        case 'Home':
          e.preventDefault();
          focusFirst();
          break;
        case 'End':
          e.preventDefault();
          focusLast();
          break;
        case 'Escape':
          e.preventDefault();
          close();
          triggerRef.current?.focus();
          break;
        case 'Tab':
          e.preventDefault();
          focusNext();
          break;
        default:
          if (isPrintableKey(e)) {
            const items = getEnabledItems();
            const chars = e.key.toLowerCase();
            const currentIdx = items.findIndex((i) => i.ref.current === active);
            const next = items.findIndex((i, idx) => {
              if (idx <= currentIdx) return false;
              const text = normalizeText(i.ref.current?.textContent);
              return text.startsWith(chars);
            });
            if (next !== -1) {
              items[next].ref.current?.focus();
            } else {
              const wrap = items.findIndex((i) => {
                const text = normalizeText(i.ref.current?.textContent);
                return text.startsWith(chars);
              });
              if (wrap !== -1) items[wrap].ref.current?.focus();
            }
          }
      }
    };
    document.addEventListener('keydown', handleKeyDown);
    return () => document.removeEventListener('keydown', handleKeyDown);
  }, [isOpen, close, focusNext, focusPrev, focusFirst, focusLast, triggerRef, getEnabledItems, searchRef]);

  useEffect(() => {
    if (isOpen) {
      requestAnimationFrame(() => {
        if (searchRef.current) {
          searchRef.current.focus();
        } else {
          focusFirst();
        }
      });
    }
  }, [isOpen, focusFirst, searchRef]);

  if (!isOpen) return null;

  return createPortal(
    <div
      ref={contentRef}
      id={`${menuId}-content`}
      role="menu"
      aria-labelledby={`${menuId}-trigger`}
      style={{
        position: 'absolute',
        top: position.top,
        left: position.left,
        backgroundColor: 'white',
        border: '1px solid #ccc',
        borderRadius: 4,
        boxShadow: '0 4px 12px rgba(0,0,0,0.15)',
        minWidth: 220,
        padding: '4px 0',
        zIndex: 1000,
        outline: 'none',
      }}
    >
      {children}
    </div>,
    document.body,
  );
};

      }, 0);
    }
  };

  const bg = disabled ? 'transparent' : isOpen || isFocused ? '#f0f0f0' : 'transparent';

  return (
```

```jsx
// ==================== Fuzzy Search ====================

function fuzzyMatch(query, text) {
  query = query.toLowerCase();
  text = text.toLowerCase();
  let qi = 0;
  let ti = 0;
  const indices = [];
  while (qi < query.length && ti < text.length) {
    if (query[qi] === text[ti]) {
      indices.push(ti);
      qi++;
    }
    ti++;
  }
  if (qi !== query.length) return null;
  let score = 0;
  for (let i = 1; i < indices.length; i++) {
    score += indices[i] - indices[i - 1];
  }
  return { indices, score };
}

function highlightFuzzy(text, indices) {
  if (!indices?.length) return text;
  const parts = [];
  let last = 0;
  for (const idx of indices) {
    if (idx > last) parts.push(text.slice(last, idx));
    parts.push(
      <mark key={idx} style={{ background: 'transparent', color: '#0969da', fontWeight: 700 }}>
        {text[idx]}
      </mark>
    );
    last = idx + 1;
  }
  if (last < text.length) parts.push(text.slice(last));
  return parts;
}

const CommandPalette = ({ children, commands, onSelect, open, onOpenChange }) => {
  const [internalOpen, setInternalOpen] = useState(false);
  const isControlled = open !== undefined;
  const isOpen = isControlled ? open : internalOpen;

  const paletteId = useId();
  const inputRef = useRef(null);
  const listRef = useRef(null);
  const overlayRef = useRef(null);

  const [query, setQuery] = useState('');
  const [activeIndex, setActiveIndex] = useState(-1);

  const [recents, setRecents] = useState(() => {
    try {
      return JSON.parse(localStorage.getItem('cmd-palette-recents') || '[]');
    } catch {
      return [];
    }
  });

  const saveRecent = useCallback((cmdId) => {
    setRecents((prev) => {
      const next = [cmdId, ...prev.filter((id) => id !== cmdId)].slice(0, 10);
      localStorage.setItem('cmd-palette-recents', JSON.stringify(next));
      return next;
    });
  }, []);

  const setIsOpen = useCallback(
    (next) => {
      const value = typeof next === 'function' ? next(isOpen) : next;
      if (!isControlled) setInternalOpen(value);
      onOpenChange?.(value);
      if (value) {
        setQuery('');
        setActiveIndex(-1);
      }
    },
    [isControlled, onOpenChange, isOpen],
  );

  const close = useCallback(() => setIsOpen(false), [setIsOpen]);

  const filtered = useMemo(() => {
    if (!query.trim()) {
      const recentSet = new Set(recents);
      const recentCmds = [];
      const others = [];
      commands.forEach((cmd) => {
        if (recentSet.has(cmd.id)) recentCmds.push(cmd);
        else others.push(cmd);
      });
      recentCmds.sort((a, b) => recents.indexOf(a.id) - recents.indexOf(b.id));
      return [...recentCmds, ...others];
    }

    const scored = [];
    commands.forEach((cmd) => {
      const titleMatch = fuzzyMatch(query, cmd.title);
      const categoryMatch = cmd.category ? fuzzyMatch(query, cmd.category) : null;
      if (titleMatch || categoryMatch) {
        scored.push({
          ...cmd,
          _match: titleMatch || categoryMatch,
          _score: (titleMatch?.score ?? 999) + (categoryMatch ? 10 : 0),
        });
      }
    });
    scored.sort((a, b) => a._score - b._score);
    return scored;
  }, [query, commands, recents]);

  const grouped = useMemo(() => {
    if (!query.trim()) {
      const groups = {};
      filtered.forEach((cmd) => {
        const cat = cmd.category || 'Commands';
        if (!groups[cat]) groups[cat] = [];
        groups[cat].push(cmd);
      });
      return groups;
    }
    return { Results: filtered };
  }, [filtered, query]);

  const flatItems = useMemo(() => {
    const items = [];
    Object.entries(grouped).forEach(([category, cmds]) => {
      items.push({ type: 'group', category });
      cmds.forEach((cmd) => items.push({ type: 'item', cmd }));
    });
    return items;
  }, [grouped]);

  // Find first item index
  const firstItemIndex = useMemo(() => {
    return flatItems.findIndex((i) => i.type === 'item');
  }, [flatItems]);

  const lastItemIndex = useMemo(() => {
    for (let i = flatItems.length - 1; i >= 0; i--) {
      if (flatItems[i].type === 'item') return i;
    }
    return -1;
  }, [flatItems]);

  const selectItem = useCallback(
    (cmd) => {
      saveRecent(cmd.id);
      onSelect?.(cmd);
      close();
    },
    [onSelect, close, saveRecent],
  );

  const scrollToIndex = useCallback((index) => {
    const el = listRef.current?.querySelector(`[data-index="${index}"]`);
    el?.scrollIntoView({ block: 'nearest' });
  }, []);

  const focusNext = useCallback(() => {
    setActiveIndex((prev) => {
      let next = prev + 1;
      while (next < flatItems.length && flatItems[next]?.type === 'group') next++;
      if (next >= flatItems.length) {
        // Wrap to first item
        next = firstItemIndex;
      }
      if (next >= 0) requestAnimationFrame(() => scrollToIndex(next));
      return next >= 0 ? next : prev;
    });
  }, [flatItems, firstItemIndex, scrollToIndex]);

  const focusPrev = useCallback(() => {
    setActiveIndex((prev) => {
      let prevIdx = prev - 1;
      while (prevIdx >= 0 && flatItems[prevIdx]?.type === 'group') prevIdx--;
      if (prevIdx < 0) {
        // Wrap to last item
        prevIdx = lastItemIndex;
      }
      if (prevIdx >= 0) requestAnimationFrame(() => scrollToIndex(prevIdx));
      return prevIdx >= 0 ? prevIdx : prev;
    });
  }, [flatItems, lastItemIndex, scrollToIndex]);

  // Global hotkey: F1 (capture phase)
  useEffect(() => {
    const handleKeyDown = (e) => {
      if (e.key === 'F1') {
        e.preventDefault();
        e.stopPropagation();
        setIsOpen((current) => !current);
      }
    };
    document.addEventListener('keydown', handleKeyDown, true);
    return () => document.removeEventListener('keydown', handleKeyDown, true);
  }, [setIsOpen]);

  const value = useMemo(
    () => ({
      isOpen,
      setIsOpen,
      close,
      paletteId,
      inputRef,
      listRef,
      overlayRef,
      query,
      setQuery,
      activeIndex,
      setActiveIndex,
      flatItems,
      grouped,
      selectItem,
      focusNext,
      focusPrev,
      firstItemIndex,
      lastItemIndex,
      scrollToIndex,
      recents,
    }),
    [
      isOpen, setIsOpen, close, paletteId, query, activeIndex,
      flatItems, grouped, selectItem, focusNext, focusPrev,
      firstItemIndex, lastItemIndex, scrollToIndex, recents,
    ],
  );

  return (
    <PaletteContext.Provider value={value}>
      {children}
    </PaletteContext.Provider>
  );
};

CommandPalette.Input = function Input({ placeholder = 'Type a command or search...' }) {
  const {
    query,
    setQuery,
    focusNext,
    focusPrev,
    flatItems,
    selectItem,
    activeIndex,
    setActiveIndex,
    firstItemIndex,
    lastItemIndex,
    close,
  } = usePalette();

  const inputRef = useRef(null);

  // Auto focus input when overlay opens
  useEffect(() => {
    const timer = setTimeout(() => {
      inputRef.current?.focus();
    }, 10);
    return () => clearTimeout(timer);
  }, []);

  const handleKeyDown = (e) => {
    switch (e.key) {
      case 'ArrowDown':
        e.preventDefault();
        if (activeIndex < 0 && firstItemIndex >= 0) {
          setActiveIndex(firstItemIndex);
        } else {
          focusNext();
        }
        break;
      case 'ArrowUp':
        e.preventDefault();
        if (activeIndex < 0 && lastItemIndex >= 0) {
          setActiveIndex(lastItemIndex);
        } else {
          focusPrev();
        }
        break;
      case 'Enter':
        e.preventDefault();
        if (activeIndex >= 0) {
          const item = flatItems[activeIndex];
          if (item?.type === 'item') selectItem(item.cmd);
        } else if (firstItemIndex >= 0) {
          const item = flatItems[firstItemIndex];
          if (item?.type === 'item') selectItem(item.cmd);
        }
        break;
      case 'Home':
        e.preventDefault();
        if (firstItemIndex >= 0) setActiveIndex(firstItemIndex);
        break;
      case 'End':
        e.preventDefault();
        if (lastItemIndex >= 0) setActiveIndex(lastItemIndex);
        break;
      case 'Escape':
        e.preventDefault();
        close();
        break;
    }
  };

  return (
    <div style={{ borderBottom: '1px solid #e5e7eb', padding: '12px 16px', display: 'flex', alignItems: 'center', gap: 10 }}>
      <span style={{ color: '#666', fontSize: 18, userSelect: 'none' }}>⌘</span>
      <input
        ref={inputRef}
        type="text"
        value={query}
        onChange={(e) => {
          setQuery(e.target.value);
          setActiveIndex(-1);
        }}
        onKeyDown={handleKeyDown}
        placeholder={placeholder}
        style={{
          width: '100%',
          border: 'none',
          outline: 'none',
          fontSize: 16,
          background: 'transparent',
        }}
```

## 💡 SENIOR INSIGHTS

### Cân Nhắc Production

**1. API Design Philosophy**

```
Compound Components là về DX (Developer Experience):

Good API:
- Readable (đọc như English)
- Predictable (behavior rõ ràng)
- Flexible (customize dễ)
- Documented (examples nhiều)

Example:
<Select>              ✅ Clear structure
  <Select.Trigger />  ✅ Readable
  <Select.Options>    ✅ Predictable nesting
    <Option />        ✅ Flexible content
  </Select.Options>
</Select>

vs.

<Select              ❌ Opaque
  trigger={...}      ❌ Render props weird
  renderOptions={...}❌ Hard to customize
/>
```

**2. Performance Trade-offs**

```jsx
// Compound Components có cost:
// - Nhiều components = nhiều re-renders potential
// - Context updates = tất cả consumers re-render

// Solutions:
// 1. Memoize context value
// 2. Split context (state vs dispatch)
// 3. Use memo on expensive children

// Khi nào optimize?
// - Nhiều hơn 50 items
// - Complex rendering logic
// - Profiler shows issues

// Khi nào KHÔNG optimize?
// - Premature optimization
// - Simple components
// - No performance issues
```

**3. Library Design**

Khi thiết kế component library:

```
✅ DO:
- Document patterns clearly
- Provide TypeScript types
- Show examples (good & bad)
- Explain trade-offs
- Support both controlled/uncontrolled
- Add accessibility by default

❌ DON'T:
- Over-engineer (YAGNI)
- Break semantic HTML
- Ignore accessibility
- Assume use cases
- Make breaking changes easily
```

### Câu Hỏi Phỏng Vấn

**Junior Level:**

Q: "Compound Components là gì?"
A: Pattern cho phép components chia sẻ state ngầm định qua Context, tạo flexible API.

**Mid Level:**

Q: "Khi nào dùng Compound Components vs Props-based API?"
A:

- Compound: Complex UI, cần flexibility, component libraries
- Props-based: Simple UI, fixed structure, type-safety quan trọng

**Senior Level:**

Q: "Làm sao optimize performance của Compound Components?"
A:

- Memoize context value
- Split context (state vs dispatch)
- useCallback cho handlers
- React.memo cho expensive children
- Profile trước khi optimize

Q: "Trade-offs của Compound Components?"
A:

- ✅ Flexibility, readability, composability
- ❌ More code, potential structure errors, performance cost
- Best for: UI libraries, complex components
- Avoid for: Simple components, extreme performance needs

### War Stories

**Story 1: Tabs Performance Hell**

```
Problem: Tabs component với 100 tabs re-render tất cả khi click
Root cause: Context value không được memoize
Solution: useMemo + split context
Result: 10x faster, smooth UX

Lesson: Luôn profile với realistic data
```

**Story 2: Menu Structure Confusion**

```
Problem: Users quên đóng Menu.Content tag
Root cause: No validation
Solution: Add dev-mode warning, better docs
Result: Less support tickets

Lesson: Good DX = fewer bugs
```

**Story 3: Accessibility Lawsuit**

```
Problem: Modal không trap focus, không ESC
Root cause: Rushed implementation
Solution: Complete rewrite với a11y first
Result: Compliant, better UX

Lesson: Accessibility là requirement, không phải nice-to-have
```
